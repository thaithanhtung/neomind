Test Instructions for Node Selection

How to test:

1. Open the app at http://localhost:5174/

The app should be running on port 5174

2. Create a node

Enter a topic in the input field and press Enter to create a node

3. Click on the node

Click anywhere on the node (not on the text). You should see:

  • A blue ring around the node
  • "Text selectable" badge in the top-right corner
  • The cursor should change to a pointer when hovering over the node

4. Check console logs

Open browser DevTools (F12) and check the Console tab. You should see:

  • handleNodeClick called for node: node-xxxxx
  • CustomNode render - id: node-xxxxx selected: true

5. Try selecting text

Once the node is selected (blue ring visible), try to select text inside the node

6. Troubleshooting

If node selection isn't working:

  • Check if clicking triggers console logs
  • Check if the node has class "selected" in DevTools Elements tab
  • Look for any JavaScript errors in the console
